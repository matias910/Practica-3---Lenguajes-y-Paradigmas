## Descripción
Este proyecto implementa un **analizador sintáctico (parser)** para validar cadenas en notación **FEN**, utilizada para representar posiciones de ajedrez.

Qué hace el programa:
- Recibe una cadena FEN
- Verifica si cumple la gramática oficial FEN
- Si es válida, muestra el tablero en una interfaz gráfica
- Si es inválida, muestra un mensaje de error indicando la causa

Lenguaje de programación:
- Java

## Requisitos minimos
Para ejecutar el programa, se requiere:

- Windows 10 / macOS / Linux
- Java JDK 17 o superior
- 4 GB de memoria RAM (8 GB recomendado)
- 200 MB de almacenamiento libre
- IntelliJ IDEA o Visual Studio recomendado (opcional)
